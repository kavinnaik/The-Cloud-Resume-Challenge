Cloud Resume Challenge

Architecture of This Portfolio

This website is designed to satisfy the Cloud Resume Challenge using AWS, serverless, and Infrastructure as Code.

End-to-End AWS Architecture

The portfolio is deployed as a static website on Amazon S3, served securely via CloudFront, and mapped to a custom domain in Route 53. A serverless visitor counter uses API Gateway, Lambda, and DynamoDB.

  • S3 static website hosting
  • CloudFront CDN with HTTPS
  • Route 53 DNS and domain
  • Lambda + DynamoDB visitor counter
  • Terraform-managed Infrastructure as Code
  • GitHub Actions CI/CD pipeline
